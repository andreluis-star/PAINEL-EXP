🔑 Acesso — Sistema de Mão de Obra (Tela 2)

Fica salvo aqui só como referência para o login manual. Por segurança, todo navegador bloqueia preenchimento automático de formulário de um site de outro domínio (a Tela 2 carrega o sistema original de mão de obra, que é um domínio diferente deste painel) — isso não é uma limitação deste sistema, é uma proteção do próprio navegador contra qualquer site. Na prática: faça o login manualmente uma única vez quando a Tela 2 aparecer, usando o login/senha acima e marcando a caixa "Lembrar senha". A partir daí, o sistema mantém a sessão aberta sozinho nas próximas exibições — não será necessário logar de novo, a não ser que o navegador do painel seja configurado para limpar dados ao fechar.
